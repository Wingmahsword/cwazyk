# Viral Lab — Frontend 🧪

A quirky, premium dashboard for the Viral Reel Maker.

## Deployment to Vercel

1. **Install Vercel CLI**: `npm i -g vercel`
2. **Deploy**: Run `vercel` in this folder.
3. **Connect to Worker**: This frontend is designed to talk to your local Python pipeline. You'll need to set up a tunnel (like Ngrok) to your local FastAPI server once you've connected them.

## Aesthetics
- **Theme**: Dark Alchemy / Cyberpunk.
- **Glassmorphism**: Backdrop blur on all cards.
- **Animations**: Framer Motion for organic "cooking" transitions.
- **Typography**: Outfit & Inter.

## Development
Run the dev server:
```bash
npm install
npm run dev
```

The app will be available at [http://localhost:3000](http://localhost:3000).
